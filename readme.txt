== Description ==
Just add LOOKBOOK links into your posts and the plugin will add the Hype button for you.

At the moment no options to configure.


LOOKBOOK Embed is in no way associated with lookbook.nu.


== Installation ==
1. Upload the lookbook-embed folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
